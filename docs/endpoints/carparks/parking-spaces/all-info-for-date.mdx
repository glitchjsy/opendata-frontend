# Fetch all space info for date

Returns a list of all space information collected for a specific date, this can populate a chart.

```
GET /v1/carparks/spaces/dates/{date}
```

### Parameters
| Name | Description |
|------|-------------|
| `date` | The date to return all space usage information for. To find valid dates, see [this endpoint](/docs/endpoints/carparks/parking-spaces/list-dates). |

:::info NOTE
To retrieve current LIVE parking space totals, see [this endpoint](/docs/endpoints/carparks/parking-spaces).
:::

<TabItem value="res-schema" label="Schema">

#### Root
| Key | Type | Description |
|-----|------|-------------|
|`results`|Array|An array of `Entry`s|

#### Entry
| Key | Type | Description |
|-----|------|-------------|
|`createdAt`|String|The date/time spaces for this carpark entry were fetched and stored in the database|
|`name`|String|The human readable name of the carpark|
|`code`|String|A unique code to identify the carpark|
|`spaces`|Number|The number of empty spaces currently available|
|`status`|String|__TODO__|
|`open`|Boolean|`true` if the carpark is open, otherwise `false`|
