# Dates With Space Data Available

Returns a list of dates when parking space information was collected. This can be used to populate a date picker.

To make use of one of the dates returned, see [Fetch all space info for date](/docs/endpoints/carparks/parking-spaces/all-info-for-date).

```
GET /v1/carparks/spaces/dates
```
